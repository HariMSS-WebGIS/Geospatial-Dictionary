  <h2 style="color: #38bdf8; font-family: 'Outfit', sans-serif; font-size: 1.4rem; margin-top: 24px; margin-bottom: 12px; font-weight: 700;">✍️ 2. Adding a New Term Manually</h2>
  <p style="color: #cbd5e1; margin-bottom: 12px;">
    If you want to contribute a term from your own knowledge, click the ➕ <strong style="color: #38bdf8;">"+ Add Term"</strong> button in the header bar and fill in these fields:
  </p>

  <ol style="padding-left: 20px; color: #cbd5e1;">
    <li style="margin-bottom: 10px;">
      <strong style="color: #f8fafc;">Term Title:</strong>
      <div style="font-size: 0.9rem; color: #94a3b8; margin-top: 2px;">
        The name of the term (e.g., <em>Multipath Error</em>). <br>
        <span style="color: #fb923c;">*Note: If you enter a title that already exists, the form will display a warning dropdown letting you switch to Edit Mode instead.*</span>
      </div>
    </li>
    <li style="margin-bottom: 10px;">
      <strong style="color: #f8fafc;">Definition:</strong>
      <div style="font-size: 0.9rem; color: #94a3b8; margin-top: 2px;">A clear, accurate explanation of what the term means. Focus on geospatial accuracy.</div>
    </li>
    <li style="margin-bottom: 10px;">
      <strong style="color: #f8fafc;">Category Stream:</strong>
      <div style="font-size: 0.9rem; color: #94a3b8; margin-top: 2px;">Select the category that fits best (e.g., choose <strong>GNSS</strong> for GPS/satellite concepts, or <strong>GIS</strong> for coordinate reference systems).</div>
    </li>
    <li style="margin-bottom: 10px;">
      <strong style="color: #f8fafc;">Aliases (Synonyms):</strong>
      <div style="font-size: 0.9rem; color: #94a3b8; margin-top: 2px;">Any other names this term is known by, separated by commas (e.g. <em>Global Positioning System, NAVSTAR</em> for GPS).</div>
    </li>
    <li style="margin-bottom: 10px;">
      <strong style="color: #f8fafc;">Related Terms:</strong>
      <div style="font-size: 0.9rem; color: #94a3b8; margin-top: 2px;">List associated concepts, separated by commas, so users can find them easily (e.g. <em>RTK, GNSS, Galileo</em>).</div>
    </li>
    <li style="margin-bottom: 10px;">
      <strong style="color: #ef4444; font-size: 0.95rem;">🖼️ Upload Illustration (Mandatory File Upload):</strong>
      <div style="font-size: 0.9rem; color: #fca5a5; margin-top: 4px; font-weight: bold; border: 1px dashed rgba(239, 68, 68, 0.4); padding: 8px; border-radius: 6px; background: rgba(239, 68, 68, 0.02);">
        You must upload an image or diagram that fits the term. Adding an image/diagram is mandatory to ensure definition cards are easy to understand visually for all users.
      </div>
    </li>
  </ol>
